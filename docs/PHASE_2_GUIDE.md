## Key Concepts to Understand

### APIs
- **Request** - What you ask the API
- **Response** - What the API gives back
- **Rate Limiting** - How often you can call
- **Error Handling** - What to do when things fail

### Prompting
- **System Prompt** - Instructions for the AI's behavior
- **User Prompt** - The actual question/task
- **Temperature** - How creative vs. factual
- **Max Tokens** - Length of response

### AI Agents
- **Input** - Data to analyze
- **Reasoning** - How AI thinks about it
- **Output** - Decision or recommendation
- **Iteration** - Improving based on feedback
